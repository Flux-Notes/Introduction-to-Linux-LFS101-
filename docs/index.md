# 🐧 Introduction to Linux — LFS101 Notes

> *"The Linux philosophy is 'Laugh in the face of danger'. Oops. Wrong one. 'Do it yourself'. Yes, that's it."*
> — **Linus Torvalds**

Welcome to **Intro to Linux (LFS101) Notes** — a structured, beginner-friendly reference for **Linux**, covering everything from the Linux Foundation and core philosophy to bash scripting, networking, and local security principles.

## 🚀 How to Use These Notes

!!! tip "New to Linux?"
    Start from the beginning — **The Linux Foundation → Philosophy → Basics and System Startup** — before moving into the command line and scripting sections.

!!! info "Have some Linux background?"
    Jump straight to **Command Line Operations**, **Text Manipulation**, or **Bash Scripting** depending on what you're revisiting.

!!! warning "Keep in mind"
    Every topic includes real commands, expected outputs, and practical context — run the commands yourself as you read for the best learning experience.

---

## 🛠️ Run Locally

```bash
# Clone the repo
git clone https://github.com/Flux-Notes/Introduction-to-Linux-LFS101-.git
cd Introduction-to-Linux-LFS101-

# Create virtual environment
python -m venv venv
source venv/bin/activate   # Mac/Linux
venv\Scripts\activate      # Windows

# Install dependencies
pip install mkdocs mkdocs-material

# Serve locally
mkdocs serve
```

Open [http://127.0.0.1:8000](http://127.0.0.1:8000) in your browser.
